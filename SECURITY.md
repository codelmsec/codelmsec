# Security Policy

## Scope

This repository contains research code accompanying CodeLMSec paper (https://arxiv.org/abs/2302.04012). The code is provided for reproducibility and research use.

## Supported Versions

Only the current version of the repository is considered supported.

No formal long term security support or maintenance is guaranteed.

## Reporting Security Issues

Please use GitHub Private Vulnerability Reporting for this repository.

Go to the repository Security tab, open Advisories, and select Report a vulnerability.

Please do not open a public GitHub issue for sensitive security reports.
